User API

Use Postman or any other API clients to access the below mentioned API endpoints(all requests are POST requests!!!)

1. Register a user

URL: https://atg-project-3.herokuapp.com/register

Input data format: { "email":"example@gmail.com", "username":"example", "password1":"example", "password2":"example" }

Response: {"message":"registration message"}

2. Login a user

URL: https://atg-project-3.herokuapp.com/login

Input data format: { "username":"example", "password":"example" }

Response: {"message":"login message"}

3. Forgot password

3.1. Send password reset email

URL: https://atg-project-3.herokuapp.com/password-reset-email

Input data format: { "username":"example" }

Response: {"message":"email message"}

3.2. Reseting the password

URL: https://atg-project-3.herokuapp.com/reset-password

Input data format: { "passwordResetToken":"token that you received in your email inbox", "password1":"example", "password2":"example" }

Response: {"message":"reset password message"}
